# 🚗 Smart Car Parking Management System

## 📖 Description

The Smart Car Parking Management System is an IoT-based project that automates parking management using embedded systems and smart sensors. It detects vehicle occupancy, monitors available parking spaces in real time, controls the entrance barrier automatically, and provides a live dashboard for monitoring parking status.

## ✨ Features

- 🚗 Real-time vehicle detection
- 🅿️ Parking occupancy monitoring
- 🚧 Automatic barrier control
- 📊 Live dashboard
- 📡 IoT connectivity

## 🛠️ Technologies

- ESP32 / STM32
- Arduino IDE
- MQTT
- HTML
- CSS
- JavaScript
